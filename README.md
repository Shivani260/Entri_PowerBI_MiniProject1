# Social Media Usage Analysis using Power BI Dashboard

## 📊 Project Overview

This project explores the complex relationships between individuals' personal demographics, lifestyle factors, and their Instagram usage patterns. It analyzes a comprehensive dataset of **5,000 users** to uncover actionable insights into how social media consumption impacts well-being, productivity, and health.

The analysis integrates three key dimensions:
- **Personal/Demographic** data (age, gender, country, income, employment, etc.)
- **Lifestyle/Health** indicators (sleep, stress, happiness, BMI, blood pressure, work-life balance)
- **Social Media Behavior** metrics (daily active minutes, sessions, engagement, content preferences, reels/stories consumption)

The final deliverable is an **interactive Power BI dashboard** that enables dynamic exploration of usage patterns, correlations, and risk profiles.

## 🎯 Objectives

- Understand distinct user segments based on Instagram usage and engagement levels
- Analyze correlations between social media usage and lifestyle/health outcomes
- Identify demographic and socio-economic factors influencing usage patterns
- Highlight high-risk user profiles (e.g., high stress, poor sleep, hypertension)
- Provide an interactive dashboard for stakeholders (researchers, marketers, public health analysts)

## 📁 Dataset

- **Source**: Kaggle (Synthetic Instagram Usage & Lifestyle dataset)
- **Records**: 5,000 users
- **Files**:
  - `Personal_Cleaned_data.xlsx`
  - `Lifestyle_Cleaned_data.xlsx`
  - `SocialMedia_Cleaned_data.xlsx`
- **Key Features**: 50+ columns after cleaning and feature engineering

## 🛠️ Tools & Technologies

- **Microsoft Excel**: Data cleaning, feature engineering, and data segmentation
- **Power BI Desktop**: Data modeling, DAX measures, interactive visualizations, and dashboard development
- **Power Query**: Advanced data transformation
